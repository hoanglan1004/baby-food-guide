## 1. Overview

### 1.1 Purpose

정적 사이트(v1)를 **동적 AI 전문가 시스템(v2)**으로 업그레이드한다.

핵심 변화:
- **정적 → 동적**: 고정 데이터 → 하윤이 상황에 맞춰 매일 새로운 추천
- **일방적 → 양방향**: 읽기만 → 질문하면 답변
- **단발성 → 성장형**: 매번 초기화 → 메모리로 지속 학습

### 1.2 Background

기존 인프라 활용:
- 텔레그램 MCP 플러그인 (이미 활성화, 봇 토큰 보유)
- NotebookLM CLI v0.3.4 (이유식 노트북 소스 40+)
- Claude Code Schedule (12개 cron task 이미 운영 중)
- 메모리 시스템 (memory.jsonl JSONL 형식)

### 1.3 핵심 설계 원칙: "함께 성장하는 AI"

```
[하윤이 성장]          [AI 성장]
───────────           ──────────
5개월 (시작 전)  →    기본 지식 로드
6개월 (초기)     →    첫 반응 학습, 선호도 파악
7개월 (중기)     →    패턴 분석, 맞춤 추천 시작
9개월 (후기)     →    알레르기 이력 축적, 개인화 완성
12개월 (완료)    →    완전 맞춤형 영양 설계

메모리 = 하윤이의 성장 일기 + AI의 학습 노트
자기 복기 = "내가 추천한 게 맞았나?" 검증
```

## 2. Scope

### 2.1 In Scope (Phase 1+2)

- [ ] 텔레그램 봇: 홍란님 질문 → Claude + NotebookLM 전문가 답변
- [ ] 매일 아침 7시 자동 전송: 오늘의 이유식 + 레시피 + 유튜브 링크
- [ ] 메모리 시스템: 하윤이 프로필, 식재료 반응, 추천 이력
- [ ] 자기 복기: 이전 추천 vs 실제 반응 비교 → 학습
- [ ] 일지 분석: 기록 기반 문제점/패턴 자동 분석
- [ ] 유튜브 요리법 검색: 레시피에 맞는 동영상 링크 자동 첨부

### 2.2 Out of Scope (v2에서는 제외)

- 사이트 data.js 자동 업데이트 (Phase 4로 미룸)
- 음성 인식/답변 (텍스트만)
- 다른 아기 지원 (하윤이 전용)

## 3. Architecture

### 3.1 시스템 구조

```
홍란님 (텔레그램)
    ↕ 질문/답변
텔레그램 MCP (Claude Code 내장)
    ↕
Claude Sonnet (처리 엔진)
    ├── NotebookLM CLI → 전문가 소스 조회
    ├── 메모리 JSON → 하윤이 기록 읽기/쓰기
    ├── YouTube 검색 → 요리법 영상 링크
    └── 일지 분석 → 패턴/문제점 감지
    ↕
Schedule (Cron) → 매일 7시 자동 실행
```

### 3.3 자기 복기 시스템

```
[매주 일요일 자동 실행]

1. 이번 주 추천 이력 조회
2. 실제 반응(일지)과 비교
   - 추천 → 잘 먹음: 정확도 +1, 패턴 강화
   - 추천 → 거부: 원인 분석, 대안 기록
   - 추천 → 알레르기: 즉시 기록, 해당 식재료 제외
3. 정확도 계산 (추천 적중률)
4. 다음 주 추천에 학습 결과 반영
5. 민호님에게 주간 리포트 전송
```

## 4. 기능 상세

### 4.1 텔레그램 봇 기능

| 기능 | 트리거 | 응답 |
|------|--------|------|
| **일반 질문** | 아무 텍스트 | NotebookLM 소스 기반 전문가 답변 |
| **오늘의 이유식** | "오늘 뭐 먹여?" | 월령+일지 기반 맞춤 식단 + 레시피 + 유튜브 |
| **반응 기록** | "잘 먹었어" / "거부" / "알레르기" | 일지에 기록 + 메모리 업데이트 |
| **주간 분석** | "이번 주 어땠어?" | 5색 균형, 영양 분석, 개선점 |
| **식재료 질문** | "OO 먹여도 돼?" | 월령 확인 + 알레르기 이력 + 전문가 답변 |
| **레시피 요청** | "OO 어떻게 만들어?" | 레시피 + 유튜브 요리법 영상 링크 |

### 4.2 매일 아침 7시 자동 전송

```

🍚 오늘의 이유식
├ 오전 10시: 쌀미음 + 당근퓨레
├ 양: 미음 5숟가락 + 당근 2숟가락
└ 🆕 당근 Day 2/3 (어제 잘 먹었어요!)

👩‍🍳 당근 퓨레 만들기

━━━━━━━━━━━━━━━━━━━━━━━━

```
📊 하윤이 이유식 주간 리포트 — 4월 3주차
━━━━━━━━━━━━━━━━━━━━━━━━

├ 총 7회 이유식
├ 😋 잘 먹음: 5회 (71%)
├ 😐 보통: 1회
├ 🙅 거부: 1회 (시금치)
└ 🚨 알레르기: 0회

🔍 AI 분석
├ 시금치 거부 → 다음 주 브로콜리로 대체 시도
├ 고구마를 특히 좋아함 → 새 식재료 섞을 때 활용
└ 오전 10시가 가장 잘 먹는 시간

🤖 자기 복기 (AI 정확도)
├ 이번 주 추천 적중률: 86%
├ 개선점: 시금치 대신 브로콜리 권장했어야 함
└ 학습: 시금치 → 대체 채소 우선 추천으로 변경

━━━━━━━━━━━━━━━━━━━━━━━━
